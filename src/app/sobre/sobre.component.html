Sobre o Gerenciador de Contatos

O Gerenciador de Contatos é uma aplicação desenvolvida para facilitar o cadastro, organização e busca de contatos pessoais e profissionais. O objetivo é oferecer uma experiência simples e intuitiva para que você tenha todas as suas informações importantes.

Este projeto foi criado por Vitor Soler, aplicando tecnologias como Angular e Spring Boot.

Aqui você pode:

  • Cadastrar e gerenciar contatos favoritos
  • Pesquisar contatos
  • Visualizar detalhes completos de cada contato
  • Editar e remover contatos a qualquer momento

Este projeto é de código aberto e serve como base para estudos e futuras melhorias.
